Exercice 02: ft_putnbr

Turn-in directory : ex02/
Files to turn in  : ft_putnbr.c
Allowed functions : write

* Create a function that displays the number passed as a parameter. The
  function must be able to handle all possible values of the int type.
* Here's how it should be prototyped :

    void ft_putnbr(int nb);

* Example usage :
    - ft_putnbr(42) outputs "42".
