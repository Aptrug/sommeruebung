Exercice 03: ft_countdown

Turn-in directory : ft_countdown/
Files to turn in  : ft_countdown.c
Allowed functions : write

* Write a program that displays all digits in descending order, followed by
  a newline.

* Example:

    $> ./ft_countdown | cat -e
    9876543210$
